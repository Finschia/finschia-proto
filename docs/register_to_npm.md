# How to register package to npm

## 1. npm login
```shell
npm login
```

check the id is right.
```shell
npm whoami
```

## 2. check the latest registered package.
```shell
npm info lbmjs-types
```

## 3. publish
```shell
npm publish
```

This doc is summary of [npm Docs](https://docs.npmjs.com/creating-a-new-npm-user-account)
